EqualTo Calc
============

## 📚 About

EqualTo Calc is a Rust crate (package) that is used by Calc and EqualTo xlsx


## 🛠️ Build
To build the library

```bash
$ cargo build --release
```

## 📏 Tests

To run the tests:

```bash
$ cargo tests
```

## 🖹 API Documentation

Documentation might be generated with

```bash
$ cargo doc --no-deps
```

-------------
